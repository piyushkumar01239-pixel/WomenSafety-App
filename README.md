# 🛡️ Women Safety SOS Alert System

A native Android app that protects women in emergencies. One tap or shake the phone 3 times — it instantly sends GPS location via SMS & Email to all emergency contacts, records 30-sec audio, and offers one-tap calling to 112/100/108.

> ✅ 100% Complete | Java + SQLite | Tested on Android 15

## Features
- 🆘 One-Touch SOS with 5-second countdown
- 📳 Shake Detection (3 shakes = SOS triggered)
- 📍 GPS Location via Google Maps link
- 📱 SMS Alerts to all emergency contacts
- 📧 Email Notifications with map + audio info
- 🎙️ 30-sec Audio Recording saved to phone
- ▶️ In-app Audio Playback in Alert History
- 📞 One-tap Call to 112 / 100 / 108
- 🏥 Nearby Hospitals via Google Maps
- ✅ I'm Safe button to cancel SOS
- 👤 User Profile with medical info storage
- 🕐 Full Alert History with tap-to-map

## Technologies Used
- Java, Android Studio, SQLite
- JavaMail API (Gmail SMTP)
- Android LocationManager, SmsManager
- MediaRecorder, SensorManager (Accelerometer)
- Google Maps & Places API

## Installation
1. Clone the repository
```bash
   git clone https://github.com/piyushkumar01239-pixel/WomenSafetyApp.git
```
2. Open in Android Studio
3. Sync Gradle files
4. Add in `build.gradle (Module: app)`:
```gradle
   implementation 'com.sun.mail:android-mail:1.6.7'
   implementation 'com.sun.mail:android-activation:1.6.7'
```
5. Run on device ▶️
